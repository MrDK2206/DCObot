About HealthMate

HealthMate is an advanced medical chatbot powered by generative AI, designed to provide accurate medical information and answer health-related questions using Gale's Encyclopedia of Medicine, Second Edition as its knowledge base.

Technologies Used:
Python Flask OpenAI GPT LangChain Pinecone Docker
Project Structure:

End-to-end-Medical-Chatbot-Generative-AI-main/

  • app.py - Main Flask application
  • Data/ - Medical data resources
  • Medical_book.pdf - Reference material
  • Dockerfile - Container configuration
  • requirements.txt - Python dependencies
  • src/ - Source code modules
  • static/ - CSS and other static files
  • templates/ - HTML templates
